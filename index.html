Unity Procedural Animation

August 2022

In this Unity project, I developed a procedural animation script that is capable of animating creatures walking over uneven terrain with 2 or more limbs. The script uses inverse kinmatics and a limb controller script to animate the create. Furthermore, it can dynamically adjusts body orientation based on the position of the limbs, resulting in realistic and believable animations. Additionally, I made the script parameters easily adjustable to provide greater control over the animation. This script allows for improvements in the believability and immersion of the game world, as it allows for dynamic animations of creatures walking regardless of terrain.

Used: C#, Unity
