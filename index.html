Projects

Water Simulation

September 2023

This project is an exploration of water simulation techniques. Two techniques for water simulation were used in this project. The first is a simple Sum of Sines technique. With the Sum of Sines technique I was able to produce a highly stylisted simulation of water as well as a low fidelity simulation. The second technique explored was the FFT technique. With this technique I was able to achieve a highly realistic simulation of the ocean akin to the ocean in Sea of Thieves.

Used: C#, Unity, HLSL, Compute Shader

Modeled Grass Simulation

July 2023

This project is an attempt at rendering a full field of fully modeled grass. To do this GPU instancing is used. The grass is fully animated and highly customizable with multiple artist modifiable parameters. To enhance the performance GPU based frustum culling is implemented

Used: C#, Unity, HLSL, Compute Shader, Blender

Unity Procedural Animation

August 2022

In this Unity project, I developed a procedural animation script that is capable of animating creatures walking over uneven terrain with 2 or more limbs. The script uses inverse kinmatics and a limb controller script to animate the creature. Furthermore, it can dynamically adjusts body orientation based on the position of the limbs, resulting in realistic and believable animations. Additionally, I made the script parameters easily adjustable to provide greater control over the animation. This script allows for improvements in the believability and immersion of the game world, as it allows for dynamic animations of creatures walking regardless of terrain.

Used: C#, Unity

Unity 2D Platformer Controller
